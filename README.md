# Social-Media-Engagement-Dashboard-


## Project Overview

This project analyzes social media engagement data to help a marketing team determine which two platforms they should continue investing in for advertising.

The marketing manager provided a dataset containing engagement metrics from three social media platforms: Instagram, Facebook, and Twitter. Due to limited resources, the company is unable to manage all three platforms and needs to identify the two most effective platforms based on performance and engagement.

Using Power BI, I cleaned, analyzed, and visualized the dataset to identify trends and generate actionable insights.

## Tools Used
Power BI – Data visualization and dashboard development
Excel / Data Cleaning – Data preparation
Data Analysis – Identifying engagement patterns and performance metrics

## Dashboard Metrics

The dashboard highlights key performance indicators including:

Total Posts: 10K
Total Likes: 70M
Total Shares: 6M
Total Views: 1 Billion
Total Comments: 9M

Visualizations included in the dashboard:

Engagement rate by platform
Total shares by platform
Total likes by post type (Image, Video, Text)
Views by platform and post type
Platform performance comparison

## Key Insights
1. Instagram – Top Performing Platform

Instagram dominates most engagement metrics across the dataset.

Highest engagement rate
Highest post views
Highest likes across all post types

Because of its strong and consistent performance, Instagram is the first recommended platform for advertising investment.

2. Facebook – Second Best Platform

Although close to Twitter in some areas, Facebook performs strongly in several important metrics.

Highest number of shares
Highest number of comments
Second-highest engagement overall

This makes Facebook the second recommended platform for continued advertising.

3. Twitter – Competitive but Slightly Behind

Twitter performs well in some metrics.

Higher engagement rate than Facebook in some cases

However, it falls behind in shares, comments, and overall interaction, making it slightly less valuable for marketing investment in this scenario.

## Final Recommendation

Based on the analysis, the two platforms the company should continue investing in are:

Instagram
Facebook

While Twitter performs competitively, Instagram and Facebook provide stronger overall engagement and interaction, making them the better platforms for marketing efforts.
